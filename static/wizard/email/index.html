  1. Create a new email, activate advanced text (instead of plain text)
  2. Go to settings
  3. View your signature here
  4. Copy the signature - Copy it like any text (don't copy the code)
    • Windows/Linux: ctrl + a then ctrl + c
    • Mac: cmd + a then cmd + c
  5. Paste in signature area
  6. Save and done!
  • For Mavericks: See How to make a n html signature in Apple Mail for Mavericks Basically:
    1. Create a new signature, quit mail
    2. Go to ~/Library/Mail/V2/MailData/Signatures/, find your new signature
    3. Open it with a text editor
    4. Replace all the <body ..... </body> part with the new template code.
    5. Save
    6. Then lock the file form the finder
  • OR you can do the same than for Gmail : copy/paste the signature from browser to Mail
  • For Mountain Lion: http://code.coneybeare.net/how-to-make-and-html-signature-in-apple-mail
  • Older versions:
    1. Go to preferences, create a new signature
    2. Quit Mail (important!!)
    3. View your signature here
    4. Save as a webarchive in your Library/Mail/Signature and replace the newest existing signature
    5. Go back to to Mail and it should work :)
    6. You can then edit directly into your signature the fields like your name, title etc...(be careful with Twitter)
